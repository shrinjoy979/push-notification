# 🔔 Push Notifications Demo

A lightweight web project for sending and handling browser push notifications using `index.html`, `push.min.js`, and `serviceWorker.min.js`.

This project demonstrates how to trigger browser notifications with a minimal frontend setup.

---

## 🚀 Features

- Simple browser push notifications  
- Uses Push.js for notification handling  
- Uses Service Worker for background support  
- Minimal setup with plain HTML  
- Lightweight and easy to understand  

---

## 🛠 Tech Stack

- HTML  
- JavaScript  
- Push.js  
- Service Worker
